# Supported Formats
| Name               | Structure      | Hotkey |
| ------------------ | -------------- | ------ |
| Full               | DD_HH_MM_SS_MS | `F1`   |
| Detail             | HH_MM_SS_MS    | `F2`   |
| Standard (Default) | HH_MM_SS       | `F3`   |
| Simple             | MM_SS          | `F4`   |
| Bare               | SS             | `F5`   | 

# Abbreviations
| Abbreviation | Represents   | 
| ------------ | ------------ |
| DD           | Days         |
| HH           | Hours        |
| MM           | Minutes      |
| SS           | Seconds      |
| MS           | Milliseconds |
