# Digit Formats
## Supported Formats
| Name               | Structure      |
|--------------------|----------------|
| Full               | DD_HH_MM_SS_MS |
| Detail             | HH_MM_SS_MS    |
| Standard (Default) | HH_MM_SS       |
| Simple             | MM_SS          |
| Bare               | SS             | 

## Abbreviations
| Abbreviation | Represents   | 
|--------------|--------------|
| DD           | Days         |
| HH           | Hours        |
| MM           | Minutes      |
| SS           | Seconds      |
| MS           | Milliseconds |
