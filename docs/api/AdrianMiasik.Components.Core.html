Namespace AdrianMiasik.Components.Core

Classes

AudioMimic

A Singleton class that allows us to transfer one UnityEngine.AudioClip from an about to be hidden/destroyed gameobject and play it here instead. Limited to only 1 sound mimic at the moment.

BooleanSlider

A ThemeElement toggle in the form of a boolean slider

BooleanToggle

A ThemeElement Toggle in the form of a two-state sprite switch. (e.g. checkbox, icon / invisible)

ClickButton

A generic button that is used to interact with the software and trigger events based on user input. Supports animations, click hold curves, unity events, and sound FX (with pitch variation). Intended to be used on all our application buttons.

ConfirmationDialog

A ThemeElement associated with a prefab. Used to prompt the user with yes/no questions. Such as confirming their action that can interrupt the active running timer. Supports custom text.

DigitFormat

A ThemeElement used primarily for generating and managing our DoubleDigit's and DigitSeparator's components. Includes support for multiple layouts: See DigitFormat.SupportedFormats using our DigitFormat.Digits.

DigitSeparator

A single text component used to hold reference and set it's color.

DoubleDigit

A ThemeElement used as an interactable/lockable input field that's designed to hold two numerical values. Can be interacted by the user via the increment / decrement arrows. Intended to be used within a DigitFormat. Has two layers of selections. One for itself, another for the input field.

Sidebar

A ThemeElement page that holds SidebarRow's and deals with group selections and animations.

SidebarRow

A ThemeElement used to display an icon, and a text label. Includes ClickButton, and a spawn animation. Intended to be used by AdrianMiasik.Components.Core.SidebarRow.sidebar.

ThemeElement

A base class that is intended to be used on any component/class that needs theming (that's automatically updated), and needs reference to our main class: PomodoroTimer.

ThemeIcon

An SVG image component that flips between two sprites. One for each theme (light / dark).

TimerProgress

An abstract base timer class that is intended to be used on any component/class that needs something to happen after X couple seconds. Supports duration, and the ability to loop.

Toggle

A blank toggle that implements IColorHook. (See ThemeElement logic) Intended to be used by BooleanToggle and BooleanSlider
