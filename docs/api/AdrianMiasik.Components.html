Namespace AdrianMiasik.Components

Classes

AboutPanel

A ThemeElement page used to display information about our application. Includes a description, social buttons (SocialButtons), version number, and a copyright disclaimer.

Background

A ThemeElement image used to pull focus off selected elements. The background is our default selection for Unity's EventSystem.

CompletionLabel

A ThemeElement used primarily to update the color of the referenced "timer completed" labels via IColorHook ColorUpdate();

ConfirmationDialog

A ThemeElement associated with a prefab. Used to prompt the user with yes/no questions. Such as confirming their action that can interrupt the active running timer. You can set the text displayed in this prompt.

CreditsBubble

A TimerProgress inheritor used for displaying the authors name of the app. Intended to minimize after a couple seconds via the base class.

DigitFormat

A ThemeElement used primarily for generating and managing our DoubleDigit's and DigitSeparator's components. Includes support for multiple layouts: See DigitFormat.SupportedFormats using our DigitFormat.Digits.

DigitSeparator

A single TMPro.TMP_Text component used to hold reference and set it's color.

DoubleDigit

A ThemeElement used as an interactable/lockable input field that's designed to hold two numerical values. Can be interacted by the user via the increment / decrement arrows. Intended to be used within a DigitFormat. Has two layers of selections. One for itself, another for the input field.
