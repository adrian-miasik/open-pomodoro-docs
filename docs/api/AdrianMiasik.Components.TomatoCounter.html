Class TomatoCounter

A ThemeElement horizontal layout group that holds Tomatoes (Pomodoros) and manipulates their state as a group. Such as completing/filling each sequential Tomato in. Includes a trashcan that is used to wipe tomato progression. Intended to be used to determine when to take long breaks.
